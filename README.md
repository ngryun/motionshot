# MotionShot Web

동영상을 업로드하면 움직이는 물체의 궤적을 한 장의 사진으로 합성해 주는 웹앱입니다.
(단종된 소니 Motion Shot 앱의 웹 버전 대체)

## 특징

- **완전 클라이언트사이드** — 서버 없음. 영상이 기기 밖으로 전송되지 않습니다.
- **데스크톱 / 모바일 대응** — 모바일 브라우저(iOS Safari, Android Chrome)에서 바로 촬영·업로드 가능.
- 파일 하나(`index.html`)가 전부라 어디든 정적 호스팅으로 배포 가능.

## 사용법

1. `index.html`을 브라우저로 열거나 정적 서버로 서빙:
   ```
   python -m http.server 8642
   ```
2. **카메라를 고정하고 찍은** 동영상 선택 (삼각대 또는 벽에 기대어 촬영 권장)
3. 프레임 수·구간·민감도 조절 후 "궤적 사진 만들기"
4. PNG/JPG로 저장

## 합성 방식

| 모드 | 원리 | 적합한 경우 |
|---|---|---|
| 자동 (움직임 감지) | 프레임별 픽셀 밝기의 **중앙값으로 배경을 추정**한 뒤, 배경과 다른 부분(움직인 물체)만 겹침 | 대부분의 경우 |
| 밝게 합성 | 픽셀별 최댓값 | 어두운 배경 + 밝은 물체 (야간 불꽃, 조명) |
| 어둡게 합성 | 픽셀별 최솟값 | 밝은 배경 + 어두운 물체 |

## 배포

정적 파일이므로 GitHub Pages, Netlify, Vercel 등에 `index.html`만 올리면 됩니다.
모바일에서 쓰려면 HTTPS 주소로 배포한 뒤 폰 브라우저에서 접속하세요.

배포 주소: https://ngryun.github.io/motionshot/
